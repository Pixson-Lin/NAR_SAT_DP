NAR_SAT_DP — 使用說明
=====================

1. 解壓本 zip 至任意資料夾。
2. 雙擊 nar_sat_dp.exe 執行（無需安裝 Python 或 7-Zip）。

命令列範例：

  nar_sat_dp.exe <輸入資料夾或檔案...> -o <輸出檔名>

拖放：可將檔案或資料夾拖到 nar_sat_dp.exe 上。

設定檔：同目錄 config\pipeline.json（可選）。
